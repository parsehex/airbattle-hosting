# Controls

All keybinds are rebindable from the in-game settings (press `H` → Keybinds).

## Movement

| Action | Default Key |
|--------|-------------|
| Forward | `↑` / `W` |
| Backward | `↓` / `S` |
| Turn Left | `←` / `A` |
| Turn Right | `→` / `D` |
| Strafe Left | `Q` |
| Strafe Right | `E` |

::: info Mohawk strafe
On the Mohawk (Copter), strafe keys activate the special ability and turn simultaneously — it's built into the ship.
:::

## Mouse Mode

Press `M` to toggle mouse mode:
- **Left Click** → Fire
- **Right Click** → Special
- **WASD** → Move

Your ship will aim toward the cursor.

## Air-Battle Additions

| Action | Default Key |
|--------|-------------|
| Quick Respawn as Ship 1–5 | `Shift` + `1`–`5` |
| Free Camera (spectating) | `C` |
| Click to Spectate | Click on a player or the minimap |

### Free Camera

While spectating, press `C` to enter free camera mode. Use the arrow keys / WASD to pan around the map freely. Press `C` again to snap back to following a player — it'll pick the nearest one to where your camera is.

## Gamepad

Gamepad support is built in. When a controller is connected:
- **D-Pad** → Movement
- **Left Stick** → Steering & thrust
- **A / X** → Fire
- **B / Y** → Special
