# About This Project: Air-Battle (-Hosting)

## Origins

My first experience with Airmash's code was StarMash, where I [forked](https://github.com/parsehex/StarWarsMod4AirMash/commits/master/) it to disable the `SWAM-PING` echo. Years later I [forked](https://github.com/parsehex/airmash-extensions/commits/master/?after=9d75a44c6e421a09cccedb375a48246844ee418e+34) the collection of StarMash extensions by fabiospampinato / `Moz://a` and made changes here and there.

In the few years following that, I began to tinker with running Airmash 100% offline so that I'd always be able to play this game that I enjoy. My first tries weren't worthwhile but I was able to get it running minimally & through good effort.

One day in fall 2024, I was hanging out in the main game and struck up a chat with a player that said they were trying to mod the server, which led me to share a bit about having done it myself before. This turned into working on a new infection mode game type for a few players, which felt like it drew overall negative attention from the rest of the players. I stopped working on the project after a few months.

While working on the new game mode, I made many of the decisions about the architecture of this meta-project. you could say in preparation for something like I'm doing now.

## My Goals

- Having somewhere that I'm welcome and not causing trouble
- To show what Airmash can be like

I'm intentionally not running multiple game mode servers, 1) because I do like ctf the most, 2) to make it clear that I'm not interested in taking over the gamever, but I do have improvements in mind. Ideally I'd like to see us integrate so that people just have a new option to play on. This extends to using the domain more generally and not just pointing to my ctf server, but that involves more than myself.

I'm aware that by doing this all I've been making decisions that won't be agreed with. I can't speak for other players but I haven't been satisfied with the state of the game for a while. "Attitude" feels like the word I want to use but seems unfair, because I 100% have an attitude too. I think in these cases it's for the best for people to co-exist without trying to force each other or themselves to be any certain way. That "live and let live" thing I heard about as a kid.

## What am I getting out of this?

- Social: I don't have a circle of friends and trying to make one
- Career: This whole project is legally encumbered, but the kind of work I'm doing for it applies elsewhere. I need the provable experience.
- I like the game: My Airmash-named reddit account u/n_airmash is only about 8 months younger than the 8yo r/airmash itself. I could do without some of the social dynamics but it's a fun game to me.

## Promises / To Be Clear

- I'm volunteering the time that I have a lot of, it's not for money and can't/won't be. If I stop liking the game then I'll probably stop working on it tbh.
- I'm doing this for myself above all. My hope is to contribute anything desired upstream to the main game, but that's not up to me alone and may be complicated to accomplish.
